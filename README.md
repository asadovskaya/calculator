# Calculator

This is a simple calculator application built with Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/calculator.git
    ```
2. Navigate to the project directory:
    ```bash
    cd calculator
    ```

## Usage

Run the calculator:
```bash
python calculator.py
```
